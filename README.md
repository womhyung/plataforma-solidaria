# 🍎 Plataforma Solidária

Uma plataforma web inteligente voltada para o combate ao desperdício de alimentos e ao apoio de famílias em situação de vulnerabilidade social.

## 🎯 Objetivo

Conectar supermercados, restaurantes, feiras, produtores rurais e instituições sociais para facilitar a doação e redistribuição de alimentos que ainda estão próprios para consumo, mas que seriam descartados.

## 🏗️ Tecnologias

### Backend
- **Node.js** com Express.js
- **MongoDB** (NoSQL) para armazenamento de dados
- **Mongoose** como ODM
- **JWT** para autenticação
- **bcryptjs** para segurança de senhas

### Frontend
- **React** com Vite
- **React Router** para roteamento
- **Tailwind CSS** para estilos
- **Axios** para requisições HTTP

## ✨ Funcionalidades

- ✅ Cadastro de doadores
- ✅ Cadastro de instituições beneficiadas
- ✅ Controle de alimentos disponíveis
- ✅ Registro de validade e quantidade dos produtos
- ✅ Sistema de localização das doações
- ✅ Histórico de entregas
- ✅ Avaliações e feedbacks das instituições
- ✅ Painel administrativo para monitoramento

## 🚀 Como Começar

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
